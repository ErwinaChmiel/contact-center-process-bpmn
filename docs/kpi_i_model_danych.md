# KPI i model danych
## Model danych
Główne tabele:
- 'calls' - połączenia,
- 'cases' - zgłoszenia,
- 'contacts' - kontakty z konsultantem,
- 'agents', 'customers' - słowniki.

  # Przykładowe KPI
  - ASA (Average Speed of Answer)
  - ATH (Average Handle Time)
  - Abandonment rate
  - Self-service rate
  - Callback rate
  - FCR (First Contact Resolution)
  - SLA (odsetki spraw rozwiązanych w czasie)
